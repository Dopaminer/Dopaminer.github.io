发表于 2020-02-22

0x00 前言

报错注入也就是利用报错信息查询到自己想要的内容。报错注入也有局限性，那就是要求页面必须对报错信息有回显。报错的方法非常多，前面写过一篇文章讲双查询注入的，理论上也属于报错注入的一种，所以在这里总结一下报错注入的多种姿势。

updatexml()：修改查询到的内容

这两个函数第二个参数的要求都是符合xpath的语法，而当不符合时，就会报错，可以利用其报错内容来显示出想要的数据。
